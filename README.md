# Placement Portal

A **Flask-based Placement Portal Web Application** that allows **Admin (Institute), Companies, and Students** to interact with the system for managing campus placements.

The system helps institutes manage **placement drives, company registrations, and student job applications** efficiently.

## Installation

### 1. Clone the Repository
git clone https://github.com/vaibhav0925/Placement_Portal.git

### 2. Navigate to the Project Folder
cd Placement_Portal

### 3. Install Dependencies
pip install flask

### 4. Run the Application
python app.py

## Features

### Admin (Institute Placement Cell)
- Pre-existing superuser of the system
- Approve or reject **company registrations**
- Approve or reject **placement drives created by companies**
- View and manage **all students, companies, and placement drives**
- Edit, delete or blacklist **students and companies**
- Search students or companies by **name or ID**

### Company
- Register and create a **company profile**
- Login after **admin approval**
- Create **placement drives (job postings)**
- View applicants for their drives
- Manage job details

### Student
- Register and create a **student profile**
- Login to the portal
- View available **placement drives**
- Apply for placement drives
- Track application status
